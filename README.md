# E-commerce website
+ In this project i have used MERN tech to build the website.

## How to run 
### To run backend 
```bash
node index.js
```

### To frontend 
```bash
npm start
```

### To access the Admin page
```bash
npm run dev
```

## NOTE
In this project There is no payment system.
